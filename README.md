#Задание 2
Возвращает сумму элементов с учётом правил:
- Все положительные числа вычитаются
- Все отрицательные числа прибавляются
- Особое правило: если число делится на 5, его значение удваивается перед применением операций

Пример: 
    Ввод: [5, -2, 3, 4, -10, 17]
    Вычисления: -10+2-3-4+20-17
    Результат: -12

#Задание 3
Класс Прямоугольник 
Действия:
- Найти периметр
- Найти пплощадь
- Проверить является ли квадратом
- Повернуть (Поменять местами ширину и высоту)

Пример: 
    Ввод: ширина = 2, длина = 3
    Результат: 
        - Периметр = 10
        - Площадь = 6
        - Это не квадрат (Проверить является ли квадратом)
        - длина = 2, ширина = 3 (Повернуть)
